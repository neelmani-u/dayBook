![Preview](dayBook.gif)
## To See and Feel what it's Like

**1. Clone this repo**

> $ git clone https://github.com/neelmani-u/dayBook

**2. Run the following command in your project directory**

> $ cd dayBook

> $ npm install

**3. How to launch**

> $ npm run both
> open http://localhost:3000/ or http://127.0.0.1:3000/ using your web browser
